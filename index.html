'Chromatical' 

Dec 2018
  • Chromatical provides users with an interactive and colorful search-interface for their favorite music topics. Up to seven different APIs are called; content links are generated on a playable piano keyboard.
  • How does one fit an entire piano keyboard inside a mobile website and still leave room for content? This was one of many questions I had to solve when designing a non-traditional user interface.
  • To ensure responsiveness across devices, the 88 piano keys overflow evenly to the left and right. As the display shrinks, the piano always remains centered; it will even rotate itself if the user changes orientation on their phone or tablet. If the user is scrolling through content, the piano will auto-hide.

'Revuer' 

June 2019
  • Can't keep track of which movies you've seen? There's an escape from the memory hole! Revuer features every movie you've ever watched, visualized for quick reference.
  • Creating a virtual theater inside a web browser? Relatively easy. Getting it to scale and generate an emotional impact on a mobile phone? That's a bit trickier.
  • Databases and spreadsheets can be dull: This is a demo/experiment that visualizes a movie database with real-world objects. Ratings and reviews are represented by a newspaper rendered entirely in CSS. Movie titles are displayed by a lighted theater marquee. Colorful tickets are used for admission (signing-in or signing-up). Elements of the Revuer theater become interactive: Chairs turn into clickable buttons, film cannisters become searchable fields, and movies are supplemented by data and images from the Open Movie Database API.
  • HTML, CSS, jQuery, JavaScript ES6, Node.js, Express.js, MongoDB, Mongoose, Mocha, Chai
